# Tubbyland 🐾
Joint Account Tracker for Ruth & Lois

## Deploy to Vercel (Free)

### Step 1 — Upload to GitHub
1. Go to https://github.com and sign in (or create free account)
2. Click **"New repository"** → name it `tubbyland` → click **Create repository**
3. Click **"uploading an existing file"**
4. Upload ALL files and folders from this zip (drag and drop the whole folder contents)
5. Click **Commit changes**

### Step 2 — Deploy on Vercel
1. Go to https://vercel.com → sign in with GitHub
2. Click **"Add New Project"**
3. Select your `tubbyland` repository
4. Leave all settings as default → click **Deploy**
5. Wait ~1 minute → you get a live URL like `tubbyland.vercel.app` ✓

### Step 3 — Add to phone home screen
**iPhone (Safari):** Open URL → Share button → "Add to Home Screen" → Add
**Android (Chrome):** Open URL → 3-dot menu → "Add to Home screen"

## Data Sync
Data is stored in Supabase and syncs across all devices in real time.
Ruth and Lois will see the same data on any device 🎉
